# Privacy Policy

**Last updated: March 23, 2026**

## Overview

This application is a personal-use tool that connects to authorized third-party APIs to retrieve the owner's own data. This privacy policy describes how data is handled.

## Data Collection

This application only accesses data belonging to its owner through authenticated API connections. No data is collected from any third parties or other users.

## Data Storage

All data retrieved by this application is stored locally on the owner's personal devices in a local database. No data is stored on external servers or cloud services beyond the original data source.

## Data Sharing

No data accessed or stored by this application is shared with, sold to, or disclosed to any third parties.

## Security

The application uses industry-standard OAuth 2.0 authentication to securely connect to APIs. API credentials and tokens are stored securely on the owner's local devices.

## Changes to This Policy

This privacy policy may be updated at any time. Changes will be reflected by updating the date at the top of this document.

## Contact

For questions about this privacy policy, contact the repository owner.
